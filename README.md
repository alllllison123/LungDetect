## 🧠 What It Does  
This AI-powered assistant analyzes chest X-rays to detect signs of lung diseases—or confirm if they’re healthy! With just a quick upload, you’ll get an instant diagnosis prediction powered by deep learning models.  

## ⚙️ Stack  
- **Backend:** Python 🐍  
- **Deployment:** Replit Cloud ☁️  
- **AI Model:** Trained using medical image datasets for accuracy and insight
